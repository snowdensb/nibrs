NIBRS Pre-Certification Tool (PCT)
  • Pre-checks NIBRS records to identify submission or validation errors
  • Accepts NIBRS data in flat file format or XML formats
  • Applies the FBIâ€™s 2019.1 NIBRS validation rules
  • Produces a list of errors following the standard FBI error file format
  • Errors may be downloaded for resolution as Excel, CSV or PDF
  • Records are automatically deleted unless explicitly saved by the user
  • PCT limitations are explained here
NIBRS to Summary Reporting Tool (N2S)
  • Must first run PCT to be able to access N2S
  • Converts NIBRS data to Summary UCR Reports
    • Return A
    • Return A By Month and Year
    • Return A Supplement
    • ASR Adult/Juvenile
    • Arson
    • Human Trafficking
    • Cargo Theft
    • SHR Negligent/Non-Negligent
  • Users may compare crime counts based on NIBRS data to prior crime counts reported under the Summary Reporting System
  • Reports are produced interactively
  • User may filter report contents
  • May be downloaded as Excel spreadsheet
  • Includes incident record search and record deletion capabilities
